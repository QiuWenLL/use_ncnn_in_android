# NCNN 人脸核验 SDK 项目

## 📝 项目概述
本项目旨在开发一个基于 Android 平台的人脸核验 SDK，将底层的 AI 推理引擎（NCNN）及核心算法封装为标准库，以便于上层 Java 业务代码快速集成与调用。

## 🎯 核心功能需求
1. **人脸核验 (1:1)**：输入人脸面部数据（图像或特征值），进行相似度比对，判断是否为同一个人。
2. **双目活体检测**：结合双目摄像头（通常为 RGB + IR 红外），进行活体防伪检测，有效防御照片、视频或面具等攻击手段，确保核验的真实性和安全性。

## ⚙️ 技术规格与运行环境
- **硬件平台**：Rockchip RK3566
- **操作系统**：Android 11
- **核心推理框架**：NCNN
- **硬件加速支持**：使用 NP（NPU 加速），充分发挥 RK 芯片的 NPU 算力提升边缘计算性能。

## 📦 交付与集成约束
- **封装形式**：将 NCNN、活体检测、人脸比对及相关前/后处理代码封装成 Android SDK（通常打包为含 JNI 接口的 `.so` 动态库及 Java 包装类 / `.aar` 文件）。
- **应用层接入**：提供标准的 Java API，使得 Android App 开发者可以在 Java 层直接引入该 SDK 并快速实现“双目活体 + 人脸 (1:1)”核验功能。
